Function : Sign In

POST/user/signin

Parameter

Key Description Example
uid User ID testaccount
upw User Password password1234

Response

{
"code": -9999,
"msg": "empty",
}
Code Msg Description
0 Success! Sign In Success
1 You did not enter an ID. Request with no ID value
2 You did not enter a PW. Request with no password value
3 This ID does not exist. The requested ID does not exist in the database
4 Invalid password. The requested password does not match the password in the database

}
